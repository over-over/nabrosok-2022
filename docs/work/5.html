Работа, выполненная в рамках серии "Журнал самовыражения". Является одной из иллюстраций для использования на страницах журнала. Изображённые персонажи режут все принятые каноны нормы, при этом относясь к этому ровно, что можно понять по их лицам. Ярко-розовая одежда, макияж, необычное сочетание одежды - обыкновенно для них. Выбранная для них поза весьма опасна для обоих. Оба поддерживают равновесие, оба важны в этой системе взглядов. Часто для самовыражения нужна поддержка - эта поддержка является основной идеей работы.
